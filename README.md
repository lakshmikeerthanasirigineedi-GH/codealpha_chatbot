# 💬 PyBuddy – Basic Python Chatbot

PyBuddy is a simple rule-based chatbot built using Python as part of my internship task at CodeAlpha.

The chatbot interacts with users through predefined conversational responses and creates a fun chatting experience.

---

# 🚀 Features

- Interactive chatbot conversation
- Dynamic name input
- Funny programming jokes
- Friendly chatbot replies
- Beginner-friendly Python project
- Handles unknown inputs gracefully

---

# 🛠 Technologies Used

- Python

```text
project/
```

# ▶️ How to Run

```bash

```

# 💬 Example Conversation

```text
You: hello
PyBuddy: Hi there 👋

You: who are you
PyBuddy: I am PyBuddy, your companion 😄

You: Keertana
```
